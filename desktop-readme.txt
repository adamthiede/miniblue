Welcome to your new computer system.

It should have everything you need.

- Web browser - Firefox
- Office programs - Libreoffice
- Media playback - VLC
- PDFs - also Firefox
- Basic text editing - this program, Xed

If you'd like a software store or any other apps, run this in the terminal:

sudo flatpak install flathub io.github.kolunmi.Bazaar -y

That will install an app store front-end for you that will let you install other apps.

This system will auto-update. Reboot about once a month. It'll also let you know when it has updated.

Happy computer-ing!
